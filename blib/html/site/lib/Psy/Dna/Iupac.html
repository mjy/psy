NAME

Psy::Dna::Iupac

__top

VERSION

Version 0.01

__top

SYNOPSIS

A static meta data construct that facilitates too/from iupac translations

Case is not checked! lowercase input will not return true

    use Psy::Dna::Iupac;
    my $foo = Psy::Dna::Iupac->new();
    ...
