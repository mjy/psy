SYNOPSIS

Contains meta-data for blocks. Internal use only for all intents and purposes. 


    use Psy::Matrix::Block
    my $foo = block->new; # see _init

label

Accessor for a -label of -blk.

literalStart

Accessor for the starting (nucleotide) position of this block, ** as originally read from the matrix **. All blocks are included.

literalEnd

As literalStart. All blocks are included.

excludedStart

Accessor for the starting position of this block, as originally read from the matrix. Bracketed blocks are _excluded_ in calculating this position.

excludedEnd

Ad excludedStart. Bracketed blocks are _excluded_ in calculating this position.

bracketed

Accessor for bracketed property. 1 == bracketed

blkLength

Length including gaps.

debugDescribe

Prints a line containing the various metadata.
